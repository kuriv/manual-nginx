常用命令

执行下面的命令，查看 Nginx 服务器版本号。

nginx -v

执行下面的命令，关闭 Nginx 服务。

sudo nginx -s stop

执行下面的命令，启动 Nginx 服务。

sudo nginx

执行下面的命令，重新加载 Nginx 服务。

sudo nginx -s reload
